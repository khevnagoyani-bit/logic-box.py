🚀 Logic Box

Logic Box is a menu-driven Python application designed to strengthen core programming skills through interactive problem-solving. Instead of performing a single task, it combines multiple beginner-friendly exercises into one simple command-line program.

✨ Why Logic Box?

This project helps users understand how different programming concepts work together by providing an interactive menu where they can choose different operations without restarting the application.

🔥 Features

- 📌 Interactive menu system
- ⭐ Generates a star ("*") triangle pattern
- 🔢 Identifies whether numbers are Even or Odd
- ➕ Calculates the total sum of a user-defined range
- 🔄 Runs continuously until the user chooses to exit
- ✅ Beginner-friendly and easy to understand

🛠️ Built With

- Python 3
- Command Line Interface (CLI)

▶️ Getting Started

1. Install Python 3.
2. Save the source code as "logic_box.py".
3. Open a terminal.
4. Run:

python logic_box.py
